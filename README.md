# AI Text Summarizer with Speech Integration

AI-powered text processing application that combines text summarization, speech-to-text conversion, text-to-speech generation, sentiment analysis, translation, PDF export, and visualization tools in a single desktop application.

## Overview

This application helps users summarize lengthy text, convert text into speech, transcribe speech into text, translate content into different languages, analyze sentiment, and generate visual insights using charts and word clouds.

## Features

- Text Summarization using NLP
- Speech-to-Text Conversion
- Text-to-Speech Generation
- Sentiment Analysis
- Language Translation
- PDF Export
- Word Cloud Visualization
- Word Frequency Analysis
- Interactive Desktop Interface

## Tech Stack

- Python
- Tkinter
- Hugging Face Transformers

## Key Functionalities

### Text Summarization
Generates concise summaries from long text using transformer-based NLP models.

### Speech-to-Text
Converts spoken audio into text using speech recognition.

### Text-to-Speech
Converts summarized content into audio output.

### Sentiment Analysis
Analyzes text and classifies emotional tone.

### Visualization
Generates word clouds and frequency distribution charts.
